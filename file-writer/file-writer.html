Why two buttons?
Browsers block direct filesystem access for security.

⬇ Download File all browsers
Uses <a download> — creates a Blob URL and triggers a download to your Downloads folder. No permission needed.

💾 Choose Save Location Chrome / Edge only
Uses the File System Access API — opens a native Save dialog so you can pick exactly where the file goes. Only available in Chromium browsers.
